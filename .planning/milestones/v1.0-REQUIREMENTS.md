# Requirements: Solar System Interactive Map

**Defined:** 2026-02-24
**Core Value:** A visually stunning, interactive 3D solar system that showcases AU Brussel's creative and technical capabilities

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### 3D Scene

- [x] **SCENE-01**: Sun rendered as emissive glowing sphere with Bloom post-processing and pulsating light
- [x] **SCENE-02**: 8 planets + Pluto rendered as textured spheres with proportional sizes
- [x] **SCENE-03**: All planets orbit the Sun with adjustable-speed orbital animation
- [x] **SCENE-04**: Each planet self-rotates on its axis with correct axial tilt
- [x] **SCENE-05**: Saturn rendered with visible ring system (torus geometry)
- [x] **SCENE-06**: Earth rendered with cloud layer overlay and orbiting Moon
- [x] **SCENE-07**: Venus rendered with atmosphere glow effect
- [x] **SCENE-08**: Jupiter rendered with 4 Galilean moons (Io, Europa, Ganymede, Callisto)
- [x] **SCENE-09**: Asteroid belt rendered between Mars and Jupiter using InstancedMesh
- [x] **SCENE-10**: Procedural starfield background using Points geometry
- [x] **SCENE-11**: Faint orbit path lines for each planet with glow on hover
- [x] **SCENE-12**: Compressed orbital distances (not real scale) for visibility

### Camera & Interaction

- [x] **CAM-01**: Free-orbit camera with OrbitControls centered on Sun by default
- [x] **CAM-02**: Click planet triggers GSAP cinematic camera fly-to (~1.5s ease)
- [x] **CAM-03**: Hover over planet shows name label, brightens orbit line, subtle scale-up
- [x] **CAM-04**: Keyboard shortcuts: 1-9 jump to planet, Escape back to overview, Space pause/resume
- [x] **CAM-05**: OrbitControls disabled during GSAP fly-to, re-enabled on completion

### UI Overlay

- [x] **UI-01**: Info panel slides in from right with planet name, tagline, 6 key stats
- [x] **UI-02**: Info panel shows atmospheric composition as animated bar chart
- [x] **UI-03**: Info panel shows 2-4 rotating fun facts per planet
- [x] **UI-04**: Info panel shows list of notable moons (clickable for major ones)
- [x] **UI-05**: Navigation sidebar (left) with all celestial bodies, click to fly-to
- [x] **UI-06**: Timeline speed control (bottom) with slider 0x-100x and play/pause
- [x] **UI-07**: Loading screen with AU Brussel branding, progress bar, and space fact
- [x] **UI-08**: "Created by AU Brussel" persistent footer badge

### Responsive

- [x] **RESP-01**: Desktop layout with sidebar + right info panel
- [x] **RESP-02**: Mobile layout with hamburger nav and bottom sheet info panel
- [x] **RESP-03**: Touch events properly isolated (panel drag doesn't rotate scene)
